Frequently Asked Questions

The tool examines page titles, meta descriptions, heading hierarchy (H1-H6), image alt attributes, schema markup, HTTP status codes, internal link health, Core Web Vitals, backlink profile, and keyword rankings. Results are compiled into a DOCX report.

The system discovers relevant keywords from your page content, then checks their Google positions using SerpApi, SearchApi, or Apify. Providers are tried in order — if one is unavailable, the next is used automatically.

Three DOCX documents: a technical analysis report (metrics and issues), a monthly summary (client-facing narrative with trends), and an action plan (prioritized tasks generated by an LLM based on the findings).

Every metric carries source attribution — where it was obtained, how it was measured, and a confidence indicator. Sections with insufficient data remain blank rather than showing fabricated values. No synthetic or estimated data appears in final deliverables.

    How the Analysis Works

    Each scan crawls the target site, gathers performance and backlink data, checks keyword positions, and compiles findings into structured deliverables.

    🧩

    On-Page Checker

    Extracts and validates title tags, meta descriptions, heading structure, image alt text, internal links, and content depth. Flags missing or duplicate elements.

    📐

    Schema Scanner

    Identifies JSON-LD, Open Graph, and Twitter Card markup. Validates schema types and reports missing or malformed structured data.

    ⚡

    Speed Metrics

    Collects Core Web Vitals (LCP, CLS, FID), accessibility scores, and page speed data from Google PageSpeed Insights for mobile and desktop.

    📈

    Position Tracker

    Automatically discovers keywords from page content and checks their Google SERP positions through multiple providers with automatic fallback.

    🔗

    Backlink Analyzer

    Examines referring domains, anchor text distribution, and authority signals. Flags toxic links and maps link velocity patterns.

    📄

    Report Builder

    Compiles all findings into three DOCX documents — a technical report, monthly summary, and action plan — with LLM-generated executive summaries.
