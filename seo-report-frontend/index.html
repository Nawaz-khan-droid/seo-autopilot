Frequently Asked Questions

The audit examines page titles, meta descriptions, heading hierarchy (H1-H6), image alt attributes, schema markup, HTTP status codes, internal link health, Core Web Vitals, backlink profile, and keyword rankings. Results are compiled into a prioritized DOCX report.

The system auto-discovers keywords from your page content, then checks their Google rankings using SerpApi, SearchApi, or Apify. Providers are tried in order — if one fails or isn't configured, the next is used automatically.

Three DOCX documents: a Technical SEO Audit (metrics and issues), a Monthly SEO Report (client-facing narrative with trends), and an SEO Action Plan (prioritized tasks generated by an LLM based on audit findings).

Every metric carries source attribution — where it was obtained, how it was measured, and a confidence indicator. Sections with insufficient data show a blank rather than fabricated values. No synthetic or estimated data is included in final reports.

    Audit Methodology

    Each audit crawls the target site, collects performance and backlink data, checks keyword rankings, and assembles findings into structured reports.

    🧩

    On-Page Analysis

    Extracts and validates title tags, meta descriptions, heading structure, image alt text, internal links, and content depth. Detects missing or duplicate elements.

    📐

    Structured Data

    Identifies JSON-LD, Open Graph, and Twitter Card markup. Validates schema types and flags missing or malformed structured data.

    ⚡

    Performance Metrics

    Collects Core Web Vitals (LCP, CLS, FID), accessibility scores, and page speed data from Google PageSpeed Insights for mobile and desktop.

    📈

    Rank Tracking

    Auto-discovers keywords from page content and checks their Google SERP positions using multiple providers with automatic fallback.

    🔗

    Backlink Profile

    Analyzes referring domains, anchor text, and authority signals. Detects toxic links and maps link velocity trends.

    📄

    Report Generation

    Compiles all findings into three DOCX documents — technical audit, monthly report, and action plan — with LLM-generated executive summaries.
